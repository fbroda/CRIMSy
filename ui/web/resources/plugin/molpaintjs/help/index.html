Help

Clear

The clear button completely wipes all atoms and bonds from the editor.

Pointer Tool

The pointer tool has three different modes of operation: Select mode, Translate mode (move) and Rotate mode:

The pointer tool operates in Select mode, when no already selected atom is under the mouse cursor when pressing the left mouse button. Within Select mode, there are 3 different submodes depending on the state of the Shift and Ctrl keys. If neither Shift nor Ctrl are pressed, prior selections are cleared and the user can make a new rectangular selection by dragging the mouse. Atoms and bonds which will be selected will be marked by red color. Upon release of the mouse button the color of selected atoms changes to green. If the Shift key is pressed while the left mouse button is pressed, no selection will be cleared. By dragging the mouse while holding the button pressed, it is possible to add additional atoms and bonds to the selection. When using the Ctrl key instead of the Shift key, it's possible to remove atoms and bonds from the selection.

Translate mode will be entered when an already selected atom is clicked. All selected atoms can be dragged around as long as the mouse button is pressed.

Rotate mode is selected, when clicking on an already selected atom while the Shift key is pressed. All selected atoms will be rotated around the z-axis when the mouse is moved along the x-axis.
